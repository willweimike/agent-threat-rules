The Agent Threat Rules Technical Steering Committee holds regular
public meetings every two weeks. Meetings are open to public
observation. The meeting link and agenda are published at least 48
hours in advance in the public ATR Calendar.

This directory contains all public minutes and agendas.

## How to observe

| Item | Detail |
|------|--------|
| Meeting platform | Google Meet (link attached to each calendar event) |
| Public calendar (embed) | [Open calendar](https://calendar.google.com/calendar/embed?src=c_a2dd390e73cd0b26fb32fd145173389ccbec312083436e3c88f7dabb35d31592%40group.calendar.google.com) |
| Public calendar (iCal subscribe) | [public/basic.ics](https://calendar.google.com/calendar/ical/c_a2dd390e73cd0b26fb32fd145173389ccbec312083436e3c88f7dabb35d31592%40group.calendar.google.com/public/basic.ics) |
| Mailing list | `atr-tsc@agentthreatrule.org` — subscription on request; [Workspace-internal archive](https://groups.google.com/a/agentthreatrule.org/g/atr-tsc). External web archive enabled once the Workspace admin opens external sharing. |
| Coordination | [GitHub Discussions: TSC Coordination](https://github.com/Agent-Threat-Rule/agent-threat-rules/discussions/categories/tsc-coordination) |
| Agenda lead time | 48 hours before the meeting |
| Minutes lead time | Within 7 days after the meeting |

## How to put an item on the agenda

Open a GitHub Issue in the canonical repository tagged `tsc-agenda`.
Format follows [TSC-ONBOARDING.md §6](../TSC-ONBOARDING.md). The next
chair picks up open `tsc-agenda` issues when assembling the agenda.

## Directory layout

```
meetings/
  README.md             ← this file, the public entry point
  AGENDA-TEMPLATE.md    ← agenda template, copy per meeting
  2026-Q3/
    TEMPLATE.md         ← minutes template, copy per meeting
    AGENDA-YYYY-MM-DD.md
    YYYY-MM-DD.md       ← minutes file
  2026-Q4/
    ...
```

Each quarter gets its own subdirectory. The minutes filename is the
meeting date (`YYYY-MM-DD.md`). The agenda filename is the meeting
date with the `AGENDA-` prefix.
