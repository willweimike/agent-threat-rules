This file is a condensed cross-reference of all binding TSC decisions
across meetings. The full audit trail, including discussion summaries
and vote tallies, lives in the per-meeting minutes under
[`meetings/YYYY-QN/`](README.md).

A decision row is added by the minute-taker at the same time the
corresponding minutes file is published.

## How this log is maintained

When publishing minutes:

1. Identify each formal decision in the meeting.
2. Add one row per decision to the table above.
3. The "Charter §5.1 class" column cites the row of the voting-threshold
   table the decision was governed by (e.g. "Routine rule ID assignment").
4. The "Vote" column records the tally as `for-against-abstain`.
5. The "Minutes" column links to the per-meeting minutes file.

If a decision is later overturned, do NOT remove the original row. Add
a new row recording the overturn, and add a "Superseded by &lt;date&gt;"
note in the "Decision" column of the original row.
