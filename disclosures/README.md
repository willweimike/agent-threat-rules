This directory holds the public conflict-of-interest disclosures
required by [TSC-CHARTER.md §7](../TSC-CHARTER.md) and
[docs/BDFL-charter.md §5](../docs/BDFL-charter.md). Each TSC member
files one disclosure per term in a file named `<github-handle>.md`,
updated within 14 days of any change in primary employment.

A disclosure MUST state, at minimum:

- The member's current primary employer
- Any equity holding exceeding 1% in any AI security or AI agent
  framework vendor
- Any prior or current commercial engagement with the ATR project
- Any board or advisor relationship that would create a conflict-of-
  interest under §7 of the Charter

Disclosures are public. Members who object to public disclosure of any
of the above MUST recuse themselves from the seat rather than redact
the disclosure.

## Filing a disclosure

1. Copy [`TEMPLATE.md`](TEMPLATE.md) to `<your-github-handle>.md`.
2. Fill in the fields. Do not omit fields; write "none" where applicable.
3. Open a PR. The PR is reviewed by another TSC member (not the
   filer) for completeness, then merged.
